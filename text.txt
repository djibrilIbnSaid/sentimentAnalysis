1. Prompting

Le prompting désigne l’action de donner une consigne ou une instruction à un modèle d’IA pour obtenir une réponse ou un résultat spécifique.
	•	Le prompt est simplement le texte ou l’instruction que l’on fournit à l’IA.
	•	L’efficacité de la réponse dépend souvent de la clarté et de la précision du prompt.

Exemples :
	•	Prompt simple : “Explique-moi ce qu’est l’intelligence artificielle.”
	•	Prompt plus complexe : “Donne-moi une définition concise de l’intelligence artificielle suivie de deux exemples concrets.”

Le prompt peut influencer le type de réponse que l’IA fournit en guidant son comportement ou son style.

2. Pre-prompting

Le pre-prompting consiste à fournir un préambule ou une instruction préalable pour orienter le modèle avant même que l’utilisateur ne donne un prompt principal.
	•	Cela permet de définir un contexte, un style de réponse, ou des limites spécifiques.
	•	En général, le pre-prompt est caché pour l’utilisateur final, surtout dans des applications pratiques.

Cas d’utilisation :
	•	Les développeurs utilisent souvent le pre-prompting pour configurer un modèle. Par exemple, un pre-prompt pourrait dire à l’IA :
	“Tu es un expert en droit français. Réponds à toutes les questions en utilisant des termes juridiques appropriés.”
	•	Ensuite, tout prompt donné par l’utilisateur sera influencé par ce contexte initial.

Exemple concret de pre-prompt suivi d’un prompt :
	•	Pre-prompt : “Agis comme un coach personnel et réponds de manière motivante.”
	•	Prompt utilisateur : “Comment améliorer ma productivité ?”
	•	Réponse attendue : Une réponse motivante et centrée sur des conseils pratiques.

Différence entre prompting et pre-prompting
	•	Prompting : L’utilisateur donne une instruction directement à l’IA.
	•	Pre-prompting : Un cadre ou contexte est défini avant l’instruction utilisateur, souvent pour influencer la manière dont le modèle répond.

En résumé, le prompting est ce que l’utilisateur fournit comme question ou instruction, tandis que le pre-prompting est une préparation invisible (ou explicite) qui oriente le comportement global de l’IA.
